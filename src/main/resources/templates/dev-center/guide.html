基本配置

添加网站

点击平台顶部导航栏的添加网站，填写网站信息及验证码属性，如下图

填写说明

  1. 网站地址支持精确与模糊匹配，精确匹配示例：www.cloudcrowd.com.cn；模糊匹配示例：*.cloudcrowd.com.cn

  2. 可根据喜好选择不同的主题图库，默认图库为：综合

  3. 可根据网站所需验证码大小，选择缩放比例，主题图片默认缩放比例为：1.5倍

缩放比例 图片尺寸
0.8倍 240px*128px
0.9倍 270px*144px
1.0倍 300px*160px
1.1倍 330px*176px
1.2倍 360px*192px
1.3倍 390px*208px
1.4倍 420px*224px
1.5倍 450px*240px
  1. 可根据喜好选择验证模式，目前支持拖拽验证和点击验证

拖拽验证

点击验证

  1. 可根据网站安全要求选择安全级别，目前支持点击验证三级安全级别：低级、中级、高级

应用平台Web端配置

VUE接入

安装
npm install vue-graphic-verify --save
使用

全局应用

main.js

import Vue from 'vue'
import {GVerifyPlugin} from 'vue-graphic-verify'

Vue.use(GVerifyPlugin, {
  apiAddr: 'https://verify.cloudcrowd.com.cn',
  webKey: 'b8db153be64749796ab48a61ebcf7a1c',
  verifyWidth: 300
})

templete file

<template>
  <graphic-verify :webKey="webKey" :verifyWidth="verifyWidth"></graphic-verify>
</template>

<script>
  export default {
    name: 'app',
    data () {
      return {
        webKey: 'b8db153be64749796ab48a61ebcf7a1c',
        verifyWidth: '450'
      }
    }
  }
</script>

局部应用

templete file

<template>
  <g-verify :webKey="webKey" :verifyWidth="verifyWidth"></g-verify>
</template>

<script>
  import gVerify from 'vue-graphic-verify'

  export default {
    name: 'app',
    data () {
      return {
        webKey: 'b8db153be64749796ab48a61ebcf7a1c',
        verifyWidth: '450'
      }
    },
    components:{
      gVerify
    }
  }
</script>

获取验证成功结果

<template>
  <graphic-verify :webKey="webKey" :verifyWidth="verifyWidth" @verify-callback="recordToken"></graphic-verify>
</template>

<script>
  export default {
    name: 'app',
    data () {
      return {
        webKey: 'b8db153be64749796ab48a61ebcf7a1c',
        verifyWidth: '450',
        verifyToken: null
      }
    },
    methods: {
      recordToken (token) {
        this.verifyToken = token
      }
    }
  }
</script>

重置验证码

<template>
  <div>
    <graphic-verify ref="gVerifyBtn" :webKey="webKey" :verifyWidth="verifyWidth"></graphic-verify>
    <button @click.prevent="reset">重置</button>
  </div>
</template>

<script>
  export default {
    name: 'app',
    data () {
      return {
        webKey: 'b8db153be64749796ab48a61ebcf7a1c',
        verifyWidth: '450'
      }
    },
    methods: {
      reset () {
        this.$refs.gVerifyBtn.reset()
      }
    }
  }
</script>
参数
参数名称 数据类型 默认值 参数说明
webKey String - webKey
verifyWidth String, Number 主题图片宽度 GraphicVerify按钮宽度
border String 1px solid rgba(51, 51, 51, 0.25) GraphicVerify按钮border样式
borderRadius String 4px GraphicVerify按钮border圆角
bgColor String rgba(255, 255, 255, 0.9) GraphicVerify按钮底色
color String #333333 GraphicVerify按钮字体颜色
successColor String #1ca21c GraphicVerify验证成功颜色
failureColor String #dd1010 GraphicVerify验证失败颜色
方法
方法名称 入参 方法说明
reset() - 重置GraphicVerify
回调方法
方法名称 出参 方法说明
verify-callback String 返回GraphicVerify验证成功token

使用

引入插件和样式

html

<html>
  <head>
    <link rel="stylesheet" type="text/css" href="https://verify.cloudcrowd.com.cn/verify.css">
    <script type="text/javascript" src="https://verify.cloudcrowd.com.cn/verify.js"></script>
  </head>
  <body>
    <div id="gVerify"></div>
    
    <script type="text/javascript">
      var gv = new GraphicVerify({
        webKey: "b8db153be64749796ab48a61ebcf7a1c",
        container: "#gVerify"
      });
    </script>
  </body>
</html>

获取验证成功结果

<html>
  <head>
    <link rel="stylesheet" type="text/css" href="https://verify.cloudcrowd.com.cn/verify.css">
    <script type="text/javascript" src="https://verify.cloudcrowd.com.cn/verify.js"></script>
  </head>
  <body>
    <div id="gVerify"></div>
    <button id="checkTokenBtn" onclick="checkToken" disabled>验证token</button>
    
    <script type="text/javascript">
      var gv = new GraphicVerify({
        webKey: "b8db153be64749796ab48a61ebcf7a1c",
        container: "#gVerify",
        successCallback: enableCheckButton
      });
      
      function enableCheckButton() {
        if (gv .verifyResult) {
          document.getElementById("checkTokenBtn").disabled = "";
        }
      }
      
      function checkToken () {
        var token = gv.verifyResult;
        # do check token...
      }
    </script>
  </body>
</html>

应用平台服务端配置

接口地址

POST:https://verify.cloudcrowd.com.cn/verify/verifyToken

接口参数

参数名称 参数类型 数据类型 参数说明
apiKey 入参 String apiKey
token 入参 String 验证码验证成功token
httpCode 出参 Integer 返回码
msg 出参 String 返回信息
success 出参 Boolean 验证结果
返回码说明
httpCode 数值说明
200 请求成功
400 请求参数错误
500 服务器出错
验证结果说明
success 结果说明
true 验证通过
false 验证失败
